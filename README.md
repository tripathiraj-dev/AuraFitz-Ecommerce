# 👔 AuraFitz - Smart E-Commerce Platform

AuraFitz ek full-stack web application hai jo modern clothing store ke liye design ki gayi hai. Isme user-friendly interface aur ek powerful admin panel hai.

### 🛠️ Tech Stack Used:
- **Backend:** Java (Advanced Java), Spring MVC
- **Frontend:** JSP, Bootstrap, JavaScript, AJAX
- **Database:** MySQL
- **Server:** XAMPP / Tomcat

### 🚀 Key Features:
- **Product Catalog:** Users can browse the latest clothing collections.
- **Admin Feedback Console:** Admins can manage user queries and feedback efficiently.
- **Responsive UI:** Fully optimized for mobile and desktop views using Bootstrap.
- **Database Integration:** Real-time data handling for products and users.

### 📂 How to Run:
1. Clone this repository.
2. Import the project into **Eclipse IDE**.
3. Setup **MySQL** database using the provided `.sql` file (if any).
4. Run on **Apache Tomcat Server**.
